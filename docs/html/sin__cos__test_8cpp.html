◆ initCosDataSetFromSinDataSet()

TestDataSet initCosDataSetFromSinDataSet ( const TestDataSet &  sinDataSet,
unsigned int  number_of_tests 
)

Initiate the cosine's test data set from the sine's data set.

cos(x) = sin(90+x). Thus, to initiate the test data set for cosine method, We will use the test data set for sin method. For each sine test data, we minus 90. That will ensure that all path coverage for the cosine method will be covered as the sine method's.

Parameters
TestDataSetThe sine's test data set
number_of_testsNumber of tests
Returns
The cosine's data set
